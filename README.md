# QR Code Generator

A simple command-line tool that generates a QR code image from any URL.

## Technologies Used
- Python
- qrcode library

## Installation
1. Clone the repo
   git clone https://github.com/yourusername/qr-generator
2. Install dependencies
   pip install qrcode[pil]

## How to Run
python qr_generator.py

Then follow the prompts:
- Paste your website link
- Choose a size (1-40)
- Choose a border thickness (1-40)

The QR code image will be saved as qr_code/youtube_qr.png
